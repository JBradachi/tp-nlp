## Introdução

Este trabalho tem como objetivo colocar em pratica conceitos
vistos em sala da matéria de processamento de linguagem natural
posteriormente será escrito um artigo sobre

Esse trabalho propõe uma arquitetura de coleta, tratamento e transformação
em um modelo ontológico (baseado previamente no modelo OD4CoT) de dados contextuais
de objetos inteligentes.
